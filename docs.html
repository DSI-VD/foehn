Typography

Fonts

The font used for the body text is Sketchnote Text and Sketchnote Square for the headings.

The fonts are served via Typekit.

Responsive typography

The typography is responsive. Based on the breakpoints,the font size and line spacing fits the viewport size.
To see it in action, juste resize your browser.
