Layout

Grid

Implementation

This 12-column, responsive grid provides structure for website content.

To use the grid, wrap each grid row in a <div> with the vd-o-layout class.

Each grid item is written semantically by its width. For example: vd-o-layout__item--1-2 = 1/2 grid item, vd-o-layout__item--2-3 = 2/3 grid item.

All grid items are full-width at small screen sizes.

Accessibility

Low-vision users should be able to increase the size of the text by up to 200 percent without breaking the layout.

Usability

Avoid text lines longer than 75 characters. Place text in narrower grid boxes to keep text lines from becoming too wide.

Ordering

You can change order of items by using vd-o-layout__item--order-*. Be carefull then. You will have to take car of the gutter by using some of the following classes:

  • vd-u-gl: add gutter on the left
  • vd-u-gr: add gutter on the right
  • vd-u-grl0: remove gutter on the left
  • vd-u-gr0: remove gutter on the right
