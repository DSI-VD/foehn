--- notes: | #### Implementation This 12-column, responsive grid provides structure for website content. To use the grid, wrap each grid row in a `
` with the `vd-o-layout` class. Each grid item is written semantically by its width. For example: `vd-o-layout__item--1-2` = 1/2 grid item, `vd-o-layout__item--2-3` = 2/3 grid item. All grid items are full-width at small screen sizes. #### Accessibility Low-vision users should be able to increase the size of the text by up to 200 percent without breaking the layout. #### Usability Avoid text lines longer than 75 characters. Place text in narrower grid boxes to keep text lines from becoming too wide. ---
