There is two breakpoint:
- `45em`
- `64em`

On wich these classes modifier are based.

| Classes modifiers | < 45 em | 45 em > < 64 em | 64em > |
|-------------------|---------|-----------------|--------|
| `.*palm*`         | yes     | no              | no     |
| `.*lap*`          | no      | yes             | no     |
| `.*lap-and-up*`   | no      | yes             | yes    |
| `.*portable*`     | yes     | yes             | no     |
| `.*desk*`         | no      | no              | yes    |
