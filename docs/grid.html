Grid

The grid is based on Bootstrap 4 grid system, with flexbox support. Please refer to their doc for more information. For more info about the flexbox grid, head here.

Responsive breakpoints

Additionnaly to the basic grid classes (xs to xl) we added a custom breakpoint xxs for smaller devices.

<312px 		Extra small
≥312px 		Small
≥576px 		Medium
≥768px 		Large
≥992px 		Extra large
